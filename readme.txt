目的是希望能夠藉由機器學習的方式
從影片分析文字、聲音、影像
來達到分析該錄像中的對象是不是自閉症
因為data是保密的關係
所以這邊只有附上兩個檔案
一個是爬蟲的檔案
一個是train分析的檔案

我主要是幫忙處理文字部分的分析

前情提要：影片中的內容是一個醫師跟有自閉症或是沒有自閉症的小孩一起閱讀繪本

爬蟲的檔案是為了搜尋網站上
